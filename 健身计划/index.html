# 周一 - 上肢训练

  1. 斜卧绳索夹胸 - 4组，每组8-12次

    • 主要训练肌肉：胸肌（尤其是胸肌的上部和中部）、前三角肌、三头肌
    • 动作指导：将卧推凳调整到大约45度的角度，斜躺在凳上，双脚平地，背部完全接触凳面。双手各握住绳索，手肘微曲。然后利用胸肌的力量，将绳索向上夹紧，使两个绳索相碰，保持一秒，然后慢慢打开双臂，还原到开始位置。

  2. 坐姿划船 - 4组，每组8-12次

    • 主要训练肌肉：背阔肌，三角肌，肱二头肌
    • 动作指导：保持背部挺直，双手握住手柄，用背部的力量拉动手柄，然后慢慢放回。

  3. 立姿杠铃推举 - 4组，每组8-12次

    • 主要训练肌肉：三角肌，三头肌
    • 动作指导：双脚打开与肩同宽，握住杠铃，用肩膀的力量将杠铃推上，然后慢慢放下。

  4. EZ杠铃二头肌弯举 - 4组，每组8-12次

    • 主要训练肌肉：肱二头肌
    • 动作指导：双手握住EZ杠铃，保持上臂不动，用二头肌的力量将杠铃弯举上来，然后慢慢放下。

  5. 双杠臂屈伸 - 4组，每组8-12次

    • 主要训练肌肉：三头肌
    • 动作指导：两手抓住杠铃，保持上身稳定，用三头肌的力量将身体推起，然后慢慢放下。

  6. 有氧运动：跑步机上快走或慢跑 20-30分钟

# 周二 - 下肢和核心训练

  1. 深蹲 - 4组，每组8-12次

    • 主要训练肌肉：大腿肌群，臀部肌肉，腿后侧肌群
    • 动作指导：双脚与肩同宽，背部挺直，用腿部的力量将身体下蹲，然后慢慢站起。

  2. 宽距硬拉 - 4组，每组8-12次

    • 主要训练肌肉：腿后侧肌群，背部，臀部
    • 动作指导：双脚比肩宽，保持背部挺直，弯腰抓住杠铃，用腿和背部的力量将杠铃提起，然后慢慢放下。

  3. 立姿提踵 - 4组，每组15-20次

    • 主要训练肌肉：小腿肌群
    • 动作指导：立直身体，用脚尖的力量将身体提起，然后慢慢放下。

  4. 平板卷腹 - 4组，每组15-20次

    • 主要训练肌肉：腹部肌群
    • 动作指导：平躺在地板上，双脚抬起形成90度角，用腹部的力量将上半身卷起，然后慢慢放下。

  5. 俄罗斯转体 - 4组，每组15-20次

    • 主要训练肌肉：腹部肌群，腰部肌肉
    • 动作指导：坐在地板上，双脚抬起，上半身微微后仰，保持这个姿势，用腹部的力量将身体从一边转到另一边。

  6. 有氧运动：跑步机上快走或慢跑 20-30分钟
